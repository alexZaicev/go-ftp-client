# GO FTP Client

![golangci-lint](https://github.com/alexZaicev/go-ftp-client/actions/workflows/golint.yaml/badge.svg)](https://github.com/alexZaicev/go-ftp-client/actions/workflows/golint.yaml/badge.svg)
